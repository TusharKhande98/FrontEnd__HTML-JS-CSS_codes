* We can design calculator using HTML & CSS.

* But if we want to make interaction with it, so it cant be possible.

* So for making interaction & adding functionality in web page we also need to use JavaScript with HTML & CSS


1. JAVA is a client-side scripting language.

2. JavaScript is a server-side scripting language.

3. HTML & CSS is not a case sensitive, But JavaScript is a case sensitive.

4. Like JAVA, C & C++ are run through command prompt, so we need compiler there. But JavaScript is an Interpreted language so it dosen't require compilation process, we can get output directly on our browser.

5. JavaScript run line by line.



* JS has a 'validation' property:- its a process to ensure that user entered required & properly formatted information throught the web form.

It has two types:-

Client-side validation - In client-side validation method, all input validations & error recovery process is carried out on the client side---> i.e. on the user's browser.

Server-side validation - In server-side validation method, all input validations & error recovery process is carried out on the server side.
